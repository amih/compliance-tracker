# NIST SP 800-53 Rev 5 — full control catalog

**Full version** — all **20 control families** with their **base controls** (control enhancements
`(N)` are omitted; get the full text with enhancements from the official NIST SP 800-53 Rev 5
catalog). This is a large federal catalog — you apply a
**baseline** (Low / Moderate / High per SP 800-53B) and tailor. For US-gov / **FedRAMP** SaaS,
`⟳` items drive the continuous-monitoring calendar. Every family's `-1` control ("Policy and
Procedures") should itself be **reviewed annually**.

Legend: checkbox = implement once & keep operating · `⟳` = recurring, set `last`/`next`.
